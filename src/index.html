SEQLite

A Light Machine Learning Plugin v1.0

配置面板


本插件支持 .fasta 或 .csv 格式输入，请务必保证所有序列已经对齐，且：
.csv文件包括两列，分别为序列数据（列名x）和表型数据（列名y）；
.fasra文件每个序列数据的描述包括量化的表型数据，以label=开头，形如
> seq1 label=1.000
AAAAAAAATTTTTTTTTCCCCCCGGGG
此外，建议选定一条野生型序列作为比较和补全基准。
强烈推荐以下配置：
单层感知器：one-hot编码，学习率0.001，epoch 5000；
卷积神经网络：VHSE编码，学习率0.0001，epoch 1000；
XGBoost：VHSE编码，estimator 2000，树深度8，学习率0.01。
强烈不推荐配置：
Transformer：无法学习到有效信息，计算开销大，仅作为一种尝试保留；
one-hot编码：0或1编码，无任何先验的生化知识，除了SLP之外不适用任何模型。
运行结束后，下方控制台将输出收敛指标。 图片窗口 将展示特征维度的总方差解释率。
